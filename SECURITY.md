# PixelForge Security Policy

## Supported Versions

Security updates are released for the latest versions of PixelForge: latest tagged version of `main` and `beta` branches.

## Reporting a Vulnerability

To report a security issue in PixelForge send an email to security@pixelforge.org.

You should receive a response within the next 30 days. If the vulnerability is accepted, a fix should be published within the next 90 days from when the issue was accepted.
